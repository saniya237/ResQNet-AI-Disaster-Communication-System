# ResQNet – AI Disaster Communication & Rescue Coordination Platform

## Overview
ResQNet is an AI-powered disaster communication system designed to help communities and rescue teams coordinate during emergencies when traditional communication infrastructure fails.

The platform enables crowdsourced incident reporting, AI-based severity detection, SOS emergency alerts, and offline peer-to-peer communication.

## Problem Statement
During disasters such as floods, earthquakes, and fires, communication networks often fail. This makes it difficult for people to report incidents and for rescue teams to receive real-time information.

## Solution
ResQNet provides a disaster intelligence platform with:

- Real-time disaster map
- Incident reporting with GPS
- AI-based severity detection
- SOS emergency alerts
- Offline communication between devices
- Rescue command dashboard

## Key Features

### Disaster Map
Displays real-time disaster incidents using GPS coordinates.

### Incident Reporting
Users report incidents with details like disaster type, injuries, and road blockages.

### ML Severity Detection
Automatically assigns severity levels to incidents.

### SOS Emergency Alerts
One-tap emergency alerts with location.

### Offline Communication
Peer-to-peer messaging using mesh networking concepts.

### AI Disaster Assistant
Provides safety guidance and information during emergencies.
